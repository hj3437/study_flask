# study_flask
 
## 소개 
메뉴판입니다. 
python 웹 프레임워크 Flask를 사용하였습니다.


## 화면 및 기능 
### 가게 목록화면
웹 서버 시작, localhost 이동, 가게 목록화면이 보여집니다.  
목록화면에서는 가게변경(추가, 삭제)을 할 수 있습니다.

### 가게 상세화면
가게를 클릭하면 상세화면 메뉴를 볼 수 있습니다.  
가게 목록화면과 유사하게 메뉴 변경을 할 수 있습니다.

### 데이터 제공기능 
데이터의 형태는 Restful API 방식중에 하나인 JSON 방식으로 제공합니다.
html, css, js는 필요없고 가게목록 및 메뉴 데이터만 필요한 경우 사용 할 수 있습니다. 


## 기술
DB: sqlite3  
ORM: sqlalchemy  
Python & Flask   
Template : Jinja
